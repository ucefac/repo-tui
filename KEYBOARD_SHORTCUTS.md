## 🎯 核心快捷键

### 导航
| 快捷键 | 功能 |
|--------|------|
| `↓` / `j` | 下一个仓库 |
| `↑` / `k` | 上一个仓库 |
| `g` | 跳到顶部 |
| `G` | 跳到底部 |
| `Ctrl+d` | 向下半屏 |
| `Ctrl+u` | 向上半屏 |

### 搜索
| 快捷键 | 功能 |
|--------|------|
| `/` | 聚焦搜索框 |
| `Esc` | 退出搜索 |
| `[字符]` | 输入搜索内容 |

### 收藏与视图 ⭐ **已修复**
| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `f` | 收藏/取消收藏当前仓库 | 原 `Shift+F` → 现在更简单 |
| `Ctrl+f` | 切换到收藏夹视图 | **新增** - toggle 行为 |

### 批量操作 ⭐ **已修复**
| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `v` | 进入/退出多选模式 | 原 `Shift+V` → 现在更简单 |
| `Space` | 选择/取消选择 | 仅在多选模式下 |
| `Ctrl+a` | 全选 | 仅在多选模式下 |

### 视图切换 ⭐ **已修复**
| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `Ctrl+r` | 切换到最近视图 | **支持 toggle 返回** |

### 全局
| 快捷键 | 功能 |
|--------|------|
| `Enter` | 打开操作菜单 |
| `m` | 切换主目录 |
| `r` | 刷新列表 |
| `t` | 主题选择器 |
| `?` | 显示帮助 |
| `q` | 退出 |

## 🎹 操作菜单快捷键

在操作菜单中：
- `1` - 在 Claude Code 中打开

- `3` - 在 VS Code 中打开

   ↑                ↑                ↑
   └── Ctrl+f ──────┘                │
   └────────────── Ctrl+r ───────────┘

## 🆕 主要变更

### 修复前 (❌)
- `f` 键无响应（需要 `Shift+F`）
- `v` 键无响应（需要 `Shift+V`）
- `Ctrl+r` 只能进入，无法返回
- `Ctrl+f` 无响应

### 修复后 (✅)
- `f` = 收藏/取消收藏
- `v` = 多选模式
- `Ctrl+r` = toggle 切换（支持返回）
- `Ctrl+f` = 切换收藏夹视图

---

## 💡 使用技巧

1. **快速收藏重要仓库**: 按 `f` 键快速标记
2. **查看收藏**: 按 `Ctrl+f` 快速切换到收藏夹视图
3. **最近工作**: 按 `Ctrl+r` 查看最近打开的仓库
4. **批量操作**: 按 `v` 进入多选模式，然后 `Space` 选择多个仓库
