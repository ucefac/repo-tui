## 键盘快捷键

### 全局
| 键 | 功能 |
|----|------|
| `Ctrl+C` | 强制退出 |
| `q` | 退出应用 |

### 导航
| 键 | 功能 |
|----|------|
| `j` / `↓` | 下一个仓库 |
| `k` / `↑` | 上一个仓库 |
| `g` | 跳转到第一个 |
| `G` | 跳转到最后一个 |
| `Ctrl+u` | 向上半页 |
| `Ctrl+d` | 向下半页 |

### 搜索
| 键 | 功能 |
|----|------|
| `/` | 聚焦搜索框 |
| `任意字符` | 开始搜索 |
| `Esc` | 清除搜索 |
| `Backspace` | 删除字符 |
| `Enter` | 确认搜索 |

### 动作菜单
| 键 | 功能 |
|----|------|
| `Enter` / `o` | 打开动作菜单 |
| `c` / `1` | cd + claude |
| `w` / `2` | 打开 WebStorm |
| `v` / `3` | 打开 VS Code |
| `f` / `4` | 打开文件管理器 |
| `Esc` / `q` | 关闭菜单 |

### 目录选择器
| 键 | 功能 |
|----|------|
| `j` / `↓` | 下一个目录 |
| `k` / `↑` | 上一个目录 |
| `Enter` | 选择目录 |
| `q` / `Esc` | 取消 |

### 其他
| 键 | 功能 |
|----|------|
| `r` | 刷新仓库列表 |
| `?` | 显示帮助 |
