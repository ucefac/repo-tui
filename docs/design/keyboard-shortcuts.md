## 🆕 主要变更

### 修复前 (❌)
- `f` 键无响应（需要 `Shift+F`）
- `v` 键无响应（需要 `Shift+V`）
- `Ctrl+r` 只能进入，无法返回
- `Ctrl+f` 无响应
- `q` 键可退出程序（子界面中也可使用）

### 修复后 (✅)
- `f` = 收藏/取消收藏
- `v` = 多选模式
- `Ctrl+r` = toggle 切换（支持返回）
- `Ctrl+f` = 切换收藏夹视图
- `Ctrl+c` = 退出程序（移除 `q` 键退出）

---

## 💡 使用技巧

1. **快速收藏重要仓库**: 按 `f` 键快速标记
2. **查看收藏**: 按 `Ctrl+f` 快速切换到收藏夹视图
3. **最近工作**: 按 `Ctrl+r` 查看最近打开的仓库
4. **批量操作**: 按 `v` 进入多选模式，然后 `Space` 选择多个仓库

## 🗑️ 删除功能

### 操作流程

```
1. 在仓库列表中选中要删除的仓库
        ↓
2. 按 `Delete` 键
        ↓
3. 显示删除确认对话框
        ↓
   ┌────┴────┐
   ↓         ↓
  按 y      按 n
   ↓         ↓
 确认删除   取消
   ↓
 显示 Toast 结果
```

### 确认对话框

```
╭─ ⚠️  Confirm Delete ───────────────────────────────────────╮
│                                                             │
│  ⚠️  Warning: This action cannot be undone                  │
│                                                             │
│  Repository: github_user_repo                               │
│  Path: ~/Projects/github/github_user_repo                   │
│                                                             │
│  This will PERMANENTLY delete the repository folder         │
│  and all its contents from your disk.                       │
│                                                             │
│            [y] Confirm Delete    [n] Cancel                 │
╰─────────────────────────────────────────────────────────────╯
```

### 安全验证 (6 层验证链)

删除操作会经过以下验证：

1. **空路径检查** - 拒绝空字符串路径
2. **绝对路径规范化** - 使用 `canonicalize()` 转换
3. **存在性检查** - 确认路径存在
4. **目录检查** - 确认是目录而非文件
5. **主目录范围检查** - 确认在家目录内
6. **Git 仓库检查** - 确认包含 `.git` 目录

### Toast 通知

| 场景 | 消息 |
|------|------|
| 删除成功 | `"{{仓库名称}}" 已被删除` |
| 删除失败 | `删除失败："{{仓库名称}}"` |
| 验证失败 | 具体错误原因 |
