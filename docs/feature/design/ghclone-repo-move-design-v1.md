# 仓库移动功能设计规范

**版本**: v1
**创建时间**: 2026-03-13
**作者**: Designer

---

## 1. 概述

本文档定义"仓库移动到主目录"功能的 UI/UX 设计规范，确保视觉一致性和交互流畅性。

## 2. 主目录选择器设计

### 2.1 布局结构

```
┌─────────────────────────────────────────┐
│       选择目标主目录 (按 Enter 确认)      │
├─────────────────────────────────────────┤
│  > /Users/xxx/Projects                  │
│    /Users/xxx/Work                      │
│    /Users/xxx/Org                       │
│                                         │
│  [↑/↓ 导航] [Enter 确认] [Esc 取消]      │
└─────────────────────────────────────────┘
```

### 2.2 尺寸规范

| 属性 | 值 |
|------|-----|
| 最小宽度 | 60 字符 |
| 最大宽度 | 80 字符 |
| 最小高度 | 8 行 |
| 最大高度 | 15 行（可滚动） |
| 边距 | 左右各 2 字符 |

### 2.3 样式规范

#### 边框样式
```rust
BorderType::Rounded  // 圆角边框
```

#### 颜色规范

| 元素 | 暗色主题 | 亮色主题 |
|------|----------|----------|
| 边框 | `#818cf8` (Indigo) | `#4f46e5` (Deep Indigo) |
| 标题 | `#e0e7ff` (Light Indigo) | `#312e81` (Dark Indigo) |
| 选中项背景 | `#4f46e5` (Indigo) | `#818cf8` (Light Indigo) |
| 选中项前景 | `#ffffff` (White) | `#1e1b4b` (Dark Navy) |
| 普通项前景 | `#94a3b8` (Slate) | `#64748b` (Slate) |

#### 标题样式
```rust
Style::default()
    .fg(Color::LightCyan)
    .bg(Color::DarkGray)
    .add_modifier(Modifier::BOLD)
```

### 2.4 交互行为

| 按键 | 行为 |
|------|------|
| `↑` / `k` | 向上导航 |
| `↓` / `j` | 向下导航 |
| `Enter` | 确认选择 |
| `Esc` | 取消操作 |
| `Home` | 跳转到第一项 |
| `End` | 跳转到最后一项 |

### 2.5 状态指示

- **选中项**: 高亮背景 + 反色文字 + `> ` 前缀
- **当前仓库所在目录**: 在路径后显示 `（当前）` 标记
- **已禁用目录**: 灰色显示，不可选择

## 3. Toast 组件设计

### 3.1 布局位置

```
┌─────────────────────────────────────────┐
│                                         │
│              (主内容区域)                │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│    ┌─────────────────────────────┐      │
│    │  ✓ repo1 移动到 Projects 成功 │      │
│    └─────────────────────────────┘      │
│                                         │
└─────────────────────────────────────────┘
```

- **水平位置**: 底部中央，距离底部 2 行
- **垂直位置**: 屏幕底部上方 2 行
- **最小宽度**: 40 字符
- **最大宽度**: 80 字符（超出截断）
- **高度**: 3 行（含边框）

### 3.2 样式规范

#### 边框样式
```rust
BorderType::Plain  //  plain 边框
```

#### 颜色规范（按类型）

| 类型 | 边框颜色 | 背景颜色 | 图标颜色 | 图标 |
|------|----------|----------|----------|------|
| Success | `#22c55e` (Green) | `#14532d` (Dark Green) | `#22c55e` | ✓ |
| Error | `#ef4444` (Red) | `#7f1d1d` (Dark Red) | `#ef4444` | ✗ |
| Warning | `#f59e0b` (Amber) | `#78350f` (Dark Amber) | `#f59e0b` | ⚠ |
| Info | `#3b82f6` (Blue) | `#1e3a8a` (Dark Blue) | `#3b82f6` | ℹ |

### 3.3 动画效果

| 阶段 | 效果 | 时长 |
|------|------|------|
| 进入 | 从底部淡入 + 向上滑动 | 200ms |
| 显示 | 静止显示 | 2-5 秒（按类型） |
| 退出 | 淡出 + 向下滑动 | 200ms |

**注意**: Ratatui 不支持动画，使用简单淡入淡出（通过连续渲染实现）

### 3.4 消息模板

```rust
// Success
"✓ {} 移动到{}成功"  // {仓库名称}, {主目录名称}

// Info (跳过)
"ℹ {}已在{}中"  // {仓库名称}, {主目录名称}

// Error
"✗ 移动失败：{}"  // {错误原因}

// Warning (冲突)
"⚠ {}已存在同名仓库"  // {目标位置}
```

## 4. 键盘交互设计

### 4.1 主流程

```
用户按下 M 键
    │
    ▼
检查选中仓库 ──(无)──▶ 显示 Info Toast: "请先选择一个仓库"
    │
   (有)
    ▼
检查主目录数量 ──(<2)──▶ 显示 Info Toast: "需要至少 2 个主目录"
    │
   (≥2)
    ▼
弹出主目录选择器
    │
    ├── Esc ──▶ 关闭选择器，返回
    │
    └── Enter ──▶ 确认选择
            │
            ▼
        检查目标是否相同 ──(是)──▶ 显示 Info Toast: "已在该目录中"
            │
           (否)
            ▼
        执行移动操作
            │
            ├── 成功 ──▶ 刷新列表，显示 Success Toast
            │
            └── 失败 ──▶ 显示 Error Toast
```

### 4.2 状态转换

```
AppState::Running
    │
    │ M 键触发
    ▼
AppState::ChoosingMoveTarget { ... }
    │
    │ Esc 取消
    │ Enter 确认
    ▼
AppState::Running (或 Moving { ... } 执行中)
```

## 7. 响应式设计

### 7.1 最小终端尺寸

| 属性 | 最小值 |
|------|--------|
| 宽度 | 80 列 |
| 高度 | 25 行 |

### 7.2 尺寸调整

- **终端宽度 < 80**: 选择器宽度自适应，最小 60 字符
- **终端高度 < 25**: 选择器高度限制为 `height - 10` 行
- **列表项超出**: 显示滚动条指示器（`▲` / `▼`）
