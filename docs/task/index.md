## 文档规范

### 创建规则

1. **Phase 文档**: `phase{n}-complete.md` / `phase{n}-plan.md`
2. **任务文档**: 使用动词开头，如 `implement-xxx.md`
3. **修复文档**: 以 `fix-` 开头

### 必须包含

- 完成状态
- 测试结果
- 下一步计划
- 时间线
